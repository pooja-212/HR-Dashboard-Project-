# HR-Dashboard-Project 

# Project Overview
The HR Attrition Dashboard is an interactive Power BI project designed to analyze employee attrition and workforce trends. It provides HR professionals with actionable insights into employee turnover based on department, job role, age group, education, salary, and years at the company, enabling data-driven decision-making to improve employee retention.

Short Description 
An interactive Power BI dashboard that analyzes employee attrition trends using HR data. It provides insights into attrition by department, job role, age group, gender, education, salary, and years at the company through dynamic visualizations and KPIs, enabling data-driven HR decision-making.

Key Technologies
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel
Data Modeling
Data Visualization
Business Intelligence (BI)
